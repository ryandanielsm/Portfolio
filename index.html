SITE DE CASAMENTO

Site de casamento desenvolvido 100% do zero, iniciando pelo protótipo no Figma, onde defini layout, identidade visual e experiência do usuário. O desenvolvimento foi feito em React, com integração ao Firebase para autenticação e armazenamento de dados, além da configuração de domínio personalizado via Registro.br. O layout utiliza Bootstrap, garantindo responsividade e boa performance.

O site possui sistema de login com código individual para cada convidado e acesso exclusivo da noiva ao dashboard. A página inicial apresenta a história do casal, carrossel de fotos do pré-wedding e navegação para as páginas de cerimônia, lista de presentes, confirmação de presença e saída.

A página de cerimônia exibe todas as informações do evento, com endereço, data, horário e integração com a API do Google Maps. A confirmação de presença permite adicionar vários convidados de uma só vez, enviando todas as informações diretamente para o dashboard.

A lista de presentes redireciona para links externos e permite a reserva de itens, registrando o nome de quem irá presentear. O dashboard centraliza confirmações de presença e presentes reservados, exibindo nomes, itens e data e horário de cada ação.
